Project Report

The Journey

My responsive website journey began with Figma, where I explored layouts using the Bootstrap grid system as my structural foundation. I focused on graphic composition, specifically repetition for consistency, emphasis to create focal points, and the strategic use of size and space to ensure visual hierarchy across devices. Reflecting on the HTML coding worksheets helped solidify the transition from a static Figma design to a dynamic, functional website, as I learned to translate the grid's structure into responsive CSS classes. This process highlighted the crucial link between design tools and front-end coding for creating effective user experiences.

From Design to Code

Figma and Bootstrap: I used Figma to design the layouts, which were built on the structure of the Bootstrap CSS grid system.

Design Philosophy

Graphic Composition: I deliberately used repetition to maintain a cohesive look, emphasis to draw attention to key elements, and the interplay of size and space to guide the user's eye and ensure readability on different screen sizes.

HTML Reflection

HTML Reflection: The HTML coding worksheets were instrumental in understanding how to implement the designs. They provided the knowledge to translate the static, responsive grid structure from Figma into functional, interactive web pages using actual code.
